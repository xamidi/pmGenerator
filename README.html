@xamidi/pmGenerator

Code extracted from deontic-logic/proof-tool (still private; readme). Can be used to generate improved versions of pmproofs.txt of the mmsolitaire project.
Exemplary generated results are available at xamidi/mmsolitaire. Eligible for high-performance computing. If you have access to a supercomputer, please consider to use this tool to further contribute to our knowledge regarding minimal proofs.

Some aspects of this tool were explicated in a proposal at the Metamath mailing list.
